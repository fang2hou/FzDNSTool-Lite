简介

一个简单的unblock-Youku DNS快速切换工具。

此项目为FzDNS项目的一个分离版本，并使用Swift重构了核心代码。

使用方式

  1. 从压缩包中将.app文件解压出来。
  2. 复制或是移动 FzDNSLite.app 到 Application（应用程序） 文件夹。
  3. 在 Lauchpad（小火箭） 打开 FzDNSLite.
  4. 在你的浏览器中打开 http://ipservice.163.com/isFromMainland.
    True -> 成功 False -> 失败.
