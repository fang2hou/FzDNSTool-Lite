简介

一个简单的unblock-Youku DNS快速切换工具。

此项目为FzDNSTool项目的一个分离版本，并使用VB.Net重构了搜寻正在联网的网络适配器的代码。

系统需求

Windows 7+

低于Windows 7需要使用者自行安装.Net Framework 4.5.2。

使用方式

  1. 从压缩包中将全部文件解压出来
  2. 使用文件夹中的FzDNSLite.exe, 点击一个解锁的按钮.
  3. 如果你之前没有关闭IPv6的功能, 那么你需要重启电脑来使修改生效.
  4. 在你的浏览器中打开 http://ipservice.163.com/isFromMainland.
    True -> 成功 False -> 失败.
